Information

This is an American Sign Language (ASL) fingerspelling practice tool. It is based on Dr. Bill Vicars' practice tool but built to use a 3D rotatable animated hand scan instead of pictures or videos.

This tool is for practice purposes only and should not be used for learning ASL. It is yet to be reviewed by a qualified source and should be viewed only as a technical demo until further notice.

New Word Alt + R
Fingerspell Space
Toggle Menu Alt + M
Increase Speed Alt + ]
Decrease Speed Alt + [
Rotate Hand Mouse Drag
